# ActiveCall

Evento disparado quando o agente recebe um evento de chamada ativa. Ou seja, quando o agente tem uma ligação ativa, o agente recebe esse evento com o callId da chamada.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. Sempre que um agente tiver uma chamada ativa, esse agente receberá esse evento.
