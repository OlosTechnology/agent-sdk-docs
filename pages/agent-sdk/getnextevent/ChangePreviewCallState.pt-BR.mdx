# ChangePreviewCallState

Evento disparado quando há uma mudança de estado na chamada de um preview. Ou seja, quando ele disca, desliga, etc, uma chamada de preview, o agente recebe esse evento com o novo estado da chamada.

Estados disponíveis:

```
- "Started"
- "CustomerConnected"
- "CustomerConnected"
- "Finished"
- "AgentConnected"
```

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. Sempre que houver uma mudança de estado na chamada de um preview, esse agente receberá esse evento. Então para fazer um teste, disque para um número, atenda uma chamada ou desligue uma chamada, e o agente receberá esse evento com o novo estado da chamada.

```js
const olosAgent = new OlosAgentWs();
olosAgent.sendPreviewCallRequest(phones);
```
