## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. Sempre que houver uma mudança de status do agente, esse agente receberá esse evento. Então para fazer um teste, mude o status do agente na interface do Olos Agent ou via API, e o agente receberá esse evento com o novo status.
