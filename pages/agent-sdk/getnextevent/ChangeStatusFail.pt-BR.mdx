# ChangeStatusFail

Evento emitido quando há uma falha na tentativa de mudança de status do agente.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Não é possível solicitar a emissão deste evento. Ele é emitido automaticamente sempre que há uma falha na tentativa de mudança de status do agente.
