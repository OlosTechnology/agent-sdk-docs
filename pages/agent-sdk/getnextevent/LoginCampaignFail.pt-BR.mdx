# LoginCampaignFail

Evento emitido quando o agente falha ao fazer login em uma campanha de voz.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Não é possível solicitar a emissão deste evento, pois ele é emitido automaticamente quando o agente falha ao fazer login em uma campanha de voz.
