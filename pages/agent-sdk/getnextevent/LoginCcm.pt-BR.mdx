# LoginCcm

Evento emitido quando o agente faz login no CCM. O login no ccm é realizado após o agente informar o passcode.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Não é possível solicitar a emissão deste evento, pois ele é emitido automaticamente quando o agente informa o passcode no softphone.
