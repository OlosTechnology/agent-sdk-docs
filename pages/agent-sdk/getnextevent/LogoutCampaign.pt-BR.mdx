# LogoutCampaign

Evento emitido quando o agente faz logout de uma campanha.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Não é possível solicitar a emissão deste evento, pois ele é emitido automaticamente quando o agente faz logout de uma campanha. Para fins de teste, o logout da campanha pode ser realizado manualmente retirando uma campanha da lista do agente na plataforma Olos.
