# Nothing

Sempre que o método getNextEvent retornar o evento do tipo Nothing, significa que não há eventos a serem processados.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

CRM não tratará este evento, pois ele é utilizado para indicar que não há eventos a serem processados. Portanto, não é necessário solicitar a emissão deste evento a partir do CRM.
