# PassCode

Evento recebido após o login do agente, contendo o código de acesso (passcode) para autenticação no softphone.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento emitido automaticamente após o login do agente, não é necessário solicitar a emissão a partir do CRM.
