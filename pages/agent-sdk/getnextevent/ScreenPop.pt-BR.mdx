# ScreenPop

Evento recebido quando há uma nova chamada para o agente, contendo informações relevantes sobre a chamada, como o número do chamador, campanha e outros dados associados à chamada. 

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento emitido automaticamente pelo sistema quando há uma nova chamada para o agente, não é necessário solicitar a emissão a partir do CRM.
