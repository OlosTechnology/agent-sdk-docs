# StopConferenceRequestAccepted

Evento disparado quando a solicitação para encerrar uma conferência é aceita, indicando que a conferência foi encerrada com sucesso.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgent = new OlosAgentWs();
olosAgent.stopConferenceRequest(); // Solicitação para encerrar a conferência
```
