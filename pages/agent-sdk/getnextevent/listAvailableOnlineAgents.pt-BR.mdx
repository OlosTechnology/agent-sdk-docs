# listAvailableOnlineAgents

Evento retorna a lista de agentes online disponíveis.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosIntegration = new OlosIntegrationWs();
olosIntegration.listAvailableOnlineAgentsByCompany({ agentId });
```
