# retrieveSupervisioningMessages

Evento retorna as mensagens do supervisor de um agente. Se é enviada ou não e se é um broadcast ou não.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgent = new OlosAgentWs();
olosAgent.retrieveSupervisioningMessages(agentId, supervisorId, startDate, endDate, broadcast, received);
```
