# AsyncMessageHistoricalResponse

Evento que retorna a solicitação dos detalhes das mensagens trocadas em uma conversação.  

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.asyncMessageHistoricalRequest(agentId, conversationId)
```
