# AsyncMessageListRepliedResponse

Evento que retorna a solicitação da lista de mensagens assíncronas que já foram respondidas e estão no aguardo de uma tréplica ou tabulação.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.asyncMessageListRepliedRequest(agentId, campaignId) // campaignId = 0, se quiser retornar todas as mensagens respondidas, independente da campanha
```
