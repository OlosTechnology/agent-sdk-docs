# AsyncMessageListResponse

Evento que retorna a solicitação da lista de mensagens assíncronas que estão no aguardo de uma réplica ou tabulação.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.asyncMessageListRequest(agentId, campaignId) // campaignId = 0, se quiser retornar todas as mensagens, independente da campanha
```
