# AsyncMessageReceived

Evento que informa a chegada de uma nova mensagem assíncrona atribuída para o agente. Para lê-la, o Agente deve enviar um comando GetAsyncMessage.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. Sempre que há uma nova mensagem assinctrona atribuida ao agente, esse agente receberá esse evento.
