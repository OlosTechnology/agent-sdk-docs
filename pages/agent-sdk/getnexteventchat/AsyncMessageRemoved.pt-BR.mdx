# AsyncMessageRemoved

Evento informa que uma mensagem onde lhe havia sido atribuída foi repassada para outro agente, devido ao tempo de inatividade excedido.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. Sempre que uma mensagem assíncrona atribuída a um agente for repassada para outro agente, esse agente receberá esse evento.
