# AsyncMessageReplyResponse

Evento informa o resultado da solicitação de resposta a uma mensagem assíncrona. O resultado pode ser sucesso ou falha, e o motivo da falha, caso haja, é informado no campo "ErrorCode" e "ErrorMsg".

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.asyncMessageSend({
  agentId,
  conversationId,
  messageId,
  to,
  subject,
  content,
  attachments,
  formatedContent,
  cc,
  bcc,
  sendType
});
```
