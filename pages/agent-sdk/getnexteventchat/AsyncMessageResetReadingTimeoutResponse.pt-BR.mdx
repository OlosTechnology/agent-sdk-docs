# AsyncMessageResetReadingTimeoutResponse

Evento informa o resultado da solicitação de reset do timeout de leitura de uma mensagem assíncrona. Assim, o Agente sinaliza que está trabalhando nela e ganha mais tempo para tratá-la e respondê-la. A resposta esperada para este comando é o evento AsyncMessageResetReadingTimeoutResponse.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.asyncMessageResetReadingTimeoutRequest(agentId, conversationId);
```
