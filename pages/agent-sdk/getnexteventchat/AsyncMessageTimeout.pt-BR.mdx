# AsyncMessageTimeout

Evento informa que excedeu o tempo limite para tratamento da mensagem que está lendo no momento. O Agente tem um tempo máximo para responder ou tabular uma mensagem assíncrona, conforme configurado na Campanha de Chat.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. evento chega automaticamente ao exceder o tempo limite para tratamento da mensagem que está lendo no momento.
