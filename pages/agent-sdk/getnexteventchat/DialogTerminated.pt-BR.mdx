# DialogTerminated

Evento sinaliza o término de um diálogo.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. Sempre que um diálogo for encerrado, esse evento será emitido automaticamente para o CRM. Mesmo após uma tabulação ou remoção do chat.
