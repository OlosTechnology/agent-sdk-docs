# DispositionResponse

Evento sinaliza a resposta de uma tabulação realizada pelo agente. Pode retornar com um código de erro que significa que houver um erro ao realizar a tabulação.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.dispositionChat(
  dialogId,
  dispositionId,
  dispositionCode || "",
  year || "0",
  month || 0,
  day || 0,
  hour || "00",
  minute || "00",
  privateAgent || false
);
```
