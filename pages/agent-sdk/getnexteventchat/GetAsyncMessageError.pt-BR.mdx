# GetAsyncMessageError

Evento sinaliza que houve um erro ao tentar buscar uma mensagem assíncrona para o agente via GetAsyncMessage, informando uma falha na solicitação.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.getAsyncMessage(agentId, conversationId); // Para testar o erro, envie um conversationId inválido.
```
