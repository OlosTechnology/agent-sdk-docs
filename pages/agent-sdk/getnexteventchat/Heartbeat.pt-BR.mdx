# Heartbeat

Evento sinaliza que a conexão entre o CRM e o canal de texto sofreu uma interrupção, ou seja, o CRM parou de enviar os sinais de vida (heartbeats) para o canal de texto.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Para testar esse evento, basta interromper o envio dos getNextEventChat do CRM para o canal de texto por um período de tempo.
