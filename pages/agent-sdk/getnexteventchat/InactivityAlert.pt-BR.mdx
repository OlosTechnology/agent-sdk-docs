# InactivityAlert

Evento sinaliza que o agente está inativo, ou seja, o agente não interagiu com a janela de atendimento por um período de tempo determinado. Esse evento vem com um dialogId e uma mensagem. Esse período de tempo e mensagem são configurados na Campanha de Chat e, ao ser atingido, o evento é emitido para o CRM. 

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

Evento não é solicidado. Sempre que o agente atingir o tempo de inatividade configurado na Campanha de Chat, esse evento é emitido para o CRM.
