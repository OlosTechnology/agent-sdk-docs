# ListDispositionResponse

Evento retorna uma lista de tabulações disponíveis para a campanha. 

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.listChatDispositions(campaignId)
```
