# ListInboxChatsResponse

Evento retorna a lista de chats dependendo do status solicitado (Fechado ou Aberto). Cada item da lista contém o id do status de envio. Isso serve para campanhas de inbox. Abaixo estão os possíveis status de envio:

```
OPENED: 1,
WAITING: 2,
SOLVED: 3,
CLOSED: 4
```

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.listInboxChats(true) // Fechados
olosAgentChannel.listInboxChats(false) // Abertos
```
