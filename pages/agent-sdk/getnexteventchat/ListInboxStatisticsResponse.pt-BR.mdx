# ListInboxStatisticsResponse

Evento retorna uma lista com a quantidade de chats por status inbox dos chats do agente.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs()
olosAgentChannel.listInboxStatistics()
```
