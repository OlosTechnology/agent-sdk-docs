# LogoutResponse

Evento disparado quando o agente solicita o logout do sistema. Não significa que o agente foi desconectado, apenas que ele solicitou o logout.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs();
const olosAgent = new OlosAgentWs();
olosAgent.coordinatedAgentLogout(); // Caso queira deslogar o agente de todos os canais.
olosAgentChannel?.agentChannelLogout(); // Caso queira deslogar apenas o canal de texto.
```
