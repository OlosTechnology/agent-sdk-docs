# ScreenPop

Evento disparado quando o agente recebe um ScreenPop de um chat. Ou seja, quando um chat é direcionado para o agente, o agente recebe o evento com dados do chat.

## Evento do tipo:
<br/>
- [x] VOZ 
- [ ] TEXTO

## Como solicitar a emissão do evento a partir do CRM

O Evento não é solicidado. Esse evento é disparado automaticamente quando um chat é direcionado para o agente.
