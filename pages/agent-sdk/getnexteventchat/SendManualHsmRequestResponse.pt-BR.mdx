# SendManualHsmRequestResponse

Evento disparado quando o agente solicita o envio de um HSM manual. Caso o evento retorne com um código de erro, houve uma falha no envio do HSM.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs();
olosAgentChannel.sendManualHsmRequest(
  agentId, 
  chatCampaignId, 
  botCampaignId, 
  phoneNumber, 
  templateId, 
  templateParams, 
  TemplateHeaderUrl, 
  message, 
  conversationId,
  extraData
)
```
