# TransferResponse

Evento disparado quando o agente solicita a transferência de um chat para outro agente ou fila. Caso o evento retorne com um código de erro, houve uma falha na transferência do chat.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs();
olosAgentChannel.transferChat(conversationId, dispositionId, dispositionCode, campaignIdDest, agentIdDest)	
```
