# Typing

Evento disparado quando o cliente começa a digitar uma mensagem para o agente. O evento é disparado apenas para webchats.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

O Evento não é solicidado. Esse evento é disparado automaticamente quando o cliente começa a digitar uma mensagem para o agente.
