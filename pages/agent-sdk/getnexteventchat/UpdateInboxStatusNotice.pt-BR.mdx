# UpdateInboxStatusNotice

Evento disparado quando há uma atualização no status de um chat do inbox do agente.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs();
olosAgentChannel.updateChatInboxStatus(conversationId, statusId)	
```
