# UpdateInboxStatusResponse

Evento disparado quando há uma solicitação à atualização do status de um chat do inbox do agente. Caso o evento retorne com um código de erro, houve uma falha na atualização do status do chat.

## Evento do tipo:
<br/>
- [ ] VOZ 
- [x] TEXTO

## Como solicitar a emissão do evento a partir do CRM

```js
const olosAgentChannel = new OlosAgentChannelWs();
olosAgentChannel.updateChatInboxStatus(conversationId, statusId)	
```
